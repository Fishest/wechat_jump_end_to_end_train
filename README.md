# Wechat_Jump_End_to_End_Train

## Prerequisite

pytorch

## How to use it

Download the dataset from [Baidu Yun](https://pan.baidu.com/s/1o9NNvqq)

Place the dataset in this folder, then

```
python train.py

```
I train this model on Ubuntu 16.04 with GTX 970
